# project
Project Overview
This project is a Python-based command-line tool designed to read, manipulate, and analyze CSV data using table-like operations.

Features
-Load and save CSV files
-Select rows based on conditions
-Project specific columns
-Sort data by selected fields
-Generate distribution counts
-Display formatted tables in the terminal

Purpose
This project demonstrates foundational programming skills including data structures, file handling, algorithm design, and modular code organization.
